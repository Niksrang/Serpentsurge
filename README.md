# Serpentine Surge

This is a modern, web-based version of the classic snake game, built with Next.js and Tailwind CSS. It features adjustable difficulty levels and an AI Strategy Lab where you can generate and observe different AI-powered snake behaviors.

- **Classic Snake Gameplay**: Control the snake to eat food and grow longer.
- **Adjustable Difficulty**: Choose between Easy, Medium, and Hard modes to match your skill level.
- **AI Strategy Generator**: Use generative AI to create unique strategies for an AI-controlled snake.
- **Real-time Visualization**: Watch the snake game unfold on a sleek, grid-based interface.
- **Responsive Design**: Play on any device, from desktop to mobile.
